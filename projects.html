Projects

I have worked on several projects throughout my time as a programmer, most of which are private. However, you might be interested in the projects that are public.

COMCAT: Automatic Software Documentation

COMCAT is an automatic software documentation tool for thorough code commenting that I developed for my research (see the Research tab above). I am currently in the process of making this tool public, but if you would like access to it, then please email me at the address at the top of this page.

GrAIter: Automated Assessment of Programming Assignments

GrAIter is a semi-automated grading tool that I developed for my research. The repository for this tool currently contains sensitive grading information, but you may contact me at the email at the top of this page if you would like to request access to a cleaned up version of this tool.

Deelzebub: A Fun Interactive Fiction Game

In my off time, I have enjoyed creating games for myself and other to play. This particular game is an interactive fiction that was entered into the 2020 IFcomp international interactive fiction competition. If you want to check it out or play online, you can do that at this link: https://ifdb.org/viewgame?id=tx41ishkacmx8lij.
